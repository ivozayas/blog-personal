PROYECTOS RECIENTES

Mokepon

Juego de peleas multijugador

Los jugadores deberán elegir a su mokepón y apareceran en el mapa, cuando dos jugadores colisionen comenzará la batalla. Entonces, cuando ambos seleccionen su secuencia de ataques se determinará el resultado del enfrentamiento. Mokepon permite jugar desde diferentes dispositivos, esto empleando express.js. Esta hecho con HTML, CSS y JavaScript.

ToDo Creator

App de tareas

ToDo Creator permite crear toDos, eliminarlos, marcarlos como completados y buscarlos según su contenido. Implementa HTML, CSS, JavaScript y React.js.

Moebius

Página de información de películas

Moebius consume la API de The Movie DB y dispone las películas según su categoría, tendencias semanales y búsqueda por títulos. Además ofrece los detalles específicos de la película seleccionada. Esta creada empleando HTML, CSS, JavaScript y React.js.

Kitties

Páginas de imágenes de gatos

Para mostrar imágenes aleatorias de gatitos, Kitties consume la api de The Cat API. Además permite la opción de guardar imágenes en favoritos y subir tus propias imágenes. Esta hecha con HTML, CSS y JavaScript
